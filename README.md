# Kushagra Mishra Personal Website

Next.js 14 App Router site with Tailwind CSS for a clean, responsive personal landing page.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run start`

## Deployment (Vercel)

1. Push your code to GitHub: `main` branch
   - `git add .`
   - `git commit -m "Prepare for Vercel deployment"`
   - `git push origin main`
2. Create Vercel project from repo: `kushagramishra1/Kushagra-Mishra`
3. Framework: `Next.js`
4. Build command: `npm run build`
5. Output directory: default (Next.js handles)
6. Optional prod deploy CLI:
   - `npx vercel --prod --confirm`
